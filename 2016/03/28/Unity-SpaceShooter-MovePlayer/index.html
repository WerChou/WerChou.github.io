Unity-SpaceShooter-MovePlayer

  • 技术博客

在构建好飞船和游戏的背景之后，我需要实现能够移动飞船的功能，这一部分将通过Script来实现。
在Unity的Script中，可以使用C#,JavaScript和Boo三种语言进行编写，在这里我选择了C#，毕竟之前使用过，也相对更熟悉点。

给对象添加Script

在Unity中，Script可以当作一个Component，在Inspector视窗中进行添加。新建的Script会默认在Assets根目录下，为了更好的
组织，我们可以新建一个“Scripts”文件夹来专门存放脚本。
在Unity中，Script的默认编辑器是MonoDeveloper，我更习惯用VS，需要注意的是，如果选择VS，需要先在Assets->Import Package中导入VS工具包，
这样可以直接在VS中进行调试。

UPDATE AND FIXEDUPDATE

初始的Script会直接设置两个函数：void Start()和void Update()，我需要用的是void FixedUpdate()。这两个有啥区别呢？
Update()
每一帧都会被调用，通常用于：

  • Moving Non-Physics Object
  • Simple Timers
  • Receiving Input
    如果每一帧间隔的时间不一致，那么每个Update()被调用的时间也会变得不同。
    FixedUpdate()
    调用时间是一致的，和Physics Step相关，被调用后，会运行必要的物理运算，所以经常用于调整物理对象例如Rigidbody
