Space-Shooter笔记——Shoting Shot

  • 技术博客
  • 读书笔记

问题

飞船的子弹造好后，还要保证随着飞船的移动而发射出去。下面对Player下的脚本进行修改，使其实现动态发射子弹的功能。

Update()

首先，把之前隐藏的Player取消勾选，编辑Player Controller脚本。
这次我需要创建Update()函数，在Unity的官方文档中，MonoBehaviour.Update() 的介绍是：

Update is called every frame, if the MonoBehaviour is enabled.
Update is the most commonly used function to implement any kind of game behaviour.
也就是每一帧更新时，执行函数内的代码。

Instantiate

官方文档中，Object.Instantiate()有两种重载形式，这里用的是：

三个参数分别为：

original ： An existing object that you want to make a copy of.
position : Position for the new object (default Vector3.zero).
rotation ： Orientation of the new object (default Quaternion.identity).
返回一个原始对象的拷贝。功能类似于直接在editor中按Ctrl+D.

新建空对象ShotSpawn

我需要一个空的对象来作为子弹的虚拟发射点，它必须和飞船的位置保持一致，所以把新建的空对象ShotSpawn
放在Player，作为它的子对象。这个时候，如果把预制块中的Bolt放在ShotSpawn，那么就可以实现子弹随飞船
移动而动态的发射了。
但是现在先把Bolt移除，我要用脚本自动生成子弹。
